# escuela-movilidad-segura-prototipo
Prototipo web educativo desarrollado como parte de la Etapa 4 del curso Proyecto de Ingeniería I (UNAD). Esta página simula un módulo digital de apoyo para las Escuelas Comunitarias de Movilidad Segura, incluyendo recursos de capacitación, normas de tránsito, videos educativos y material básico para la formación vial.
